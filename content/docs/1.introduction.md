# Components that feel like they belong

Handcrafted Vue components built with Tailwind CSS and motion-v. No npm install — just copy the code and make it yours.

## Quick Start

### 1. Run the CLI

Use the install command to add dependencies automatically.

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/hyper-text.json"
```

### 2. Copy the snippet

Grab the component code with one click.

### 3. Make it yours

Customize colors, sizes, and animations to match your brand.
