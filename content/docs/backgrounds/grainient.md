## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/grainient.json"
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color1` | `string` | `'#FF9FFC'` | First gradient color (hex). |
| `color2` | `string` | `'#5227FF'` | Second gradient color (hex). |
| `color3` | `string` | `'#B19EEF'` | Third gradient color (hex). |
| `speed` | `number` | `0.25` | Animation speed multiplier. |
| `grain-amount` | `number` | `0.1` | Film grain intensity. |
| `contrast` | `number` | `1.5` | Color contrast multiplier. |
