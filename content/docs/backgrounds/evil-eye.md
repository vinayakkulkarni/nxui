## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/evil-eye.json"
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `eye-color` | `string` | `'#FF6F37'` | Primary eye color |
| `intensity` | `number` | `1.5` | Brightness |
| `pupil-size` | `number` | `0.6` | Pupil size |
| `iris-width` | `number` | `0.25` | Iris ring width |
| `glow-intensity` | `number` | `0.35` | Outer glow |
| `scale` | `number` | `0.8` | Zoom |
| `pupil-follow` | `number` | `1.0` | Cursor tracking |
| `flame-speed` | `number` | `1.0` | Fire animation speed |
