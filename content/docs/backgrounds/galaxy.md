## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/galaxy.json"
```

## Props

| Prop               | Type     | Default     | Description               |
| ------------------ | -------- | ----------- | ------------------------- |
| `star-count`       | `number` | `2000`      | Number of stars           |
| `speed`            | `number` | `0.5`       | Animation speed           |
| `star-size`        | `number` | `2`         | Size of star points       |
| `star-color`       | `string` | `'#ffffff'` | Star color                |
| `background-color` | `string` | `'#000000'` | Background color          |
