## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/decay-card.json"
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `width` | `number` | `300` | Card width in pixels |
| `height` | `number` | `400` | Card height in pixels |
| `image` | `string` | `'https://picsum.photos/300/400?grayscale'` | Image URL |
