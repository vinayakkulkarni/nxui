## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/scroll-based-velocity.json"
```


## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | — | The text content to display with velocity-based animation |
| `default-velocity` | `number` | `5` | Default velocity multiplier for the scroll-based animation |
| `class` | `string` | — | Additional CSS classes for styling |
