## Installation

```bash
npx shadcn-vue@latest add "https://nxui.geoql.in/r/electric-border.json"
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'#5227FF'` | Color of the electric border and glow |
| `speed` | `number` | `1` | Animation speed multiplier |
| `chaos` | `number` | `0.12` | Noise distortion amplitude |
| `border-radius` | `number` | `24` | Border radius in pixels |
