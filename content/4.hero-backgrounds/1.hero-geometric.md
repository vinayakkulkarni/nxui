## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/hero-geometric.json"
```

## Usage

```vue
<script setup lang="ts">
  import HeroGeometric from '~/components/ui/HeroGeometric.vue';
</script>

<template>
  <HeroGeometric
    title="Build Beautiful UIs"
    subtitle="Animated Vue components you can copy and paste."
    badge="nxui"
  />
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | `'Components for Perfectionists'` | Main heading text |
| `subtitle` | `string` | `''` | Subtitle text below the title |
| `badge` | `string` | `'Open Source'` | Badge text in the top-left |
| `badgeIcon` | `string` | `'lucide:sparkles'` | Icon name for the badge |
| `class` | `string` | `''` | Additional CSS classes for styling |
