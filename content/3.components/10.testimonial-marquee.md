## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/testimonial-marquee.json"
```

## Usage

```vue
<script setup lang="ts">
  import TestimonialMarquee from '~/components/ui/TestimonialMarquee.vue';

  const testimonials = [
    { name: 'Alice Johnson', role: 'Designer', text: 'Amazing UI library!' },
    { name: 'Bob Smith', role: 'Developer', text: 'Smooth animations!' },
  ];
</script>

<template>
  <TestimonialMarquee class="py-6">
    <div
      v-for="t in testimonials"
      :key="t.name"
      class="mx-2 w-72 shrink-0 rounded-xl border bg-card p-4"
    >
      <p class="mb-3 text-sm text-muted-foreground">"{{ t.text }}"</p>
      <div class="text-sm font-medium">{{ t.name }}</div>
      <div class="text-xs text-muted-foreground">{{ t.role }}</div>
    </div>
  </TestimonialMarquee>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `pauseOnHover` | `boolean` | `true` | Pause animation when hovering |
| `speed` | `number` | `40` | Animation speed in pixels per second |
| `direction` | `'left' \| 'right'` | `'left'` | Direction of marquee animation |
| `class` | `string` | `''` | Additional CSS classes for styling |
