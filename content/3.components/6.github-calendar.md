## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/github-calendar.json"
```

## Usage

```vue
<script setup lang="ts">
  import GithubCalendar from '~/components/ui/GithubCalendar.vue';
</script>

<template>
  <GithubCalendar username="vinayakkulkarni" color-scheme="green" />
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `username` | `string` | `'vinayakkulkarni'` | GitHub username to fetch contribution data from |
| `year` | `number` | current year | Year to display contributions for |
| `colorScheme` | `'green' \| 'blue' \| 'purple'` | `'green'` | Color scheme for the calendar |
| `class` | `string` | `''` | Additional CSS classes for styling |
