## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/spotlight-card.json"
```

## Usage

```vue
<script setup lang="ts">
  import SpotlightCard from '~/components/ui/SpotlightCard.vue';
</script>

<template>
  <SpotlightCard class="max-w-sm p-8">
    <h3 class="mb-2 text-lg font-semibold">Spotlight Card</h3>
    <p class="text-sm text-muted-foreground">
      Hover to see the spotlight effect follow your cursor.
    </p>
  </SpotlightCard>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `spotlightColor` | `string` | `'rgba(120, 119, 198, 0.3)'` | Color of the spotlight effect |
| `borderWidth` | `number` | `1` | Width of the border in pixels |
| `borderRadius` | `number` | `16` | Border radius in pixels |
| `glowIntensity` | `number` | `0.15` | Intensity of the glow effect |
| `class` | `string` | `''` | Additional CSS classes for styling |
