## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/circuit-board.json"
```

## Usage

```vue
<script setup lang="ts">
  import CircuitBoard from '~/components/ui/CircuitBoard.vue';
</script>

<template>
  <CircuitBoard class="h-[350px] w-full rounded-xl" />
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `string` | `'#22d3ee'` | Color of the circuit board nodes |
| `nodeCount` | `number` | `30` | Number of nodes to render |
| `speed` | `number` | `1` | Animation speed multiplier |
| `glowIntensity` | `number` | `0.8` | Intensity of the glow effect |
| `class` | `string` | `''` | Additional CSS classes for styling |
