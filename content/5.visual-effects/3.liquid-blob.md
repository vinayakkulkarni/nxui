## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/liquid-blob.json"
```

## Usage

```vue
<script setup lang="ts">
  import LiquidBlob from '~/components/ui/LiquidBlob.vue';
</script>

<template>
  <LiquidBlob class="h-[350px] w-full rounded-xl bg-neutral-950">
    <div class="flex h-full items-center justify-center">
      <p class="text-lg font-medium text-white/80">Move your cursor</p>
    </div>
  </LiquidBlob>
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| color1 | `string` | `'#7c3aed'` | Primary color of the blob. |
| color2 | `string` | `'#06b6d4'` | Secondary color of the blob. |
| size | `number` | `300` | Size of the blob in pixels. |
| speed | `number` | `1` | Animation speed multiplier. |
| class | `string` | - | Additional CSS classes for the container. |
