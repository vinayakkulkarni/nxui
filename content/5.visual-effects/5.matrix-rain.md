## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/matrix-rain.json"
```

## Usage

```vue
<script setup lang="ts">
  import MatrixRain from '~/components/ui/MatrixRain.vue';
</script>

<template>
  <MatrixRain class="h-[350px] w-full rounded-xl" />
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| color | `string` | `'#0f0'` | Color of the falling characters. |
| fontSize | `number` | `14` | Font size of the characters in pixels. |
| speed | `number` | `1` | Animation speed multiplier. |
| class | `string` | - | Additional CSS classes for the container. |
