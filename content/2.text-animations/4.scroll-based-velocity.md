## Installation

```bash
npx shadcn-vue@latest add "https://nxui.dev/r/scroll-based-velocity.json"
```

## Usage

```vue
<script setup lang="ts">
  import ScrollBasedVelocity from '~/components/ui/ScrollBasedVelocity.vue';
</script>

<template>
  <ScrollBasedVelocity text="nxui — " class="text-4xl font-bold" />
</template>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| text | `string` | - | The text content to display with velocity-based animation. |
| defaultVelocity | `number` | `5` | Default velocity multiplier for the scroll-based animation. |
| class | `string` | - | Additional CSS classes for styling. |
